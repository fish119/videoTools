我想做一个windows桌面应用，来实现对指定视频的操作，包括区间剪裁功能、视频压缩、视频合并。要求如下：

# 整体要求
1. 只有一个图形化界面，分为3个tab，分别是：区间剪裁、视频压缩、视频合并
2. 默认显示的是第一个tab界面，即区间剪裁界面
3. 点击不同tab可实现在相应界面间切换
4. 界面美观、windows11风格，字体清晰
5. 最终程序可打包为windows安装程序，或可直接运行的exe程序，程序名称为：videoTools.exe
6. 这是一个windows桌面应用，不需要启动浏览器进行测试

# 区间剪裁
1. 在界面中选择需要剪裁的视频
2. 在界面中输入想剪裁的时间段（一个多行输入框即可），时间段为多个，每个都包括“开始时间”和“结束时间”，时间格式为：hh:mm:ss。输入内容样例：
	```txt
	00:02:20
	00:03:30
	
	00:04:20
	00:05:10
	
	00:20:21
	00:28:27
	```

1. 每个时间段的第一行为开始时间，第二行为结束时间，不同时间段以1个空行分隔
2. 选择输出结果的保存目录，默认为：D:\Download，同时要有按钮能够在资源管理器中打开该文件夹
3. 点击开始，自动按照输入的各个时间段的“开始时间”和“结束时间”，将所选择的视频文件剪裁成相应的多个片段
4. 剪裁完成的文件，文件名按时间段先后命名为01、02、03……
5. 处理完成后提示完成。

# 视频压缩
1. 在界面中选择需要压缩的视频，可多选
2. 压缩的视频编码为：H.265
3. 视频质量（CRF值）为：30
4. 选择输出结果的保存目录，默认为：D:\Download，同时要有按钮能够在资源管理器中打开该文件夹
5. 点击开始压缩，压缩完成的文件，文件名保持原文件名即可
6. 处理完成后提示完成。

# 视频合并
1. 在界面中选择需要压缩的视频，可多选
2. 点击开始合并，按照文件名升序合并视频文件，合并完成的文件名为marg_时间戳(格式为：YYYYMMDDhhmmss)
3. 处理完成后提示完成。
